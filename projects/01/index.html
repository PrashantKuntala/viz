Histogram of RNA Pol-II coding genes

across four gene categories as described in the Yeast Epigenome Project.

Method

  • Data for 392 protein targets were retrieved from yeastepigenome.org
  • Each target had zero or more replicates and consists of only YPD/Normal (growth condition) samples.
  • The gene counts per gene category was calculated. For targets which had replicates, gene counts were merged and duplicates were removed.
  • Then, the genecounts data was binned using d3.bin() for each gene category to generate the histogram.

How to read

  • Each bar represents a bin.
  • No.of targets in each bin is given by the bar height.
  • For example, the RP genes histogram shows that there are about 188 targets that bind to less than 10 RP genes.
